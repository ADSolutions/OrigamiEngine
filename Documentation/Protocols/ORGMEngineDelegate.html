Conforms to NSObject

Overview

The delegate of a ORGMEngine object must adopt the ORGMEngineDelegate protocol. This protocol allows you to get state change notifications and implement continious playback.

engineExpectsNextUrl:

Asks the delegate for the next track url.

- (NSURL *)engineExpectsNextUrl:(ORGMEngine *)engine

Parameters

engine

The engine object requesting this information.

Return Value

The url object to be used as a source path during playback.

Discussion

Asks the delegate for the next track url.

This method provides continious playback functionality. When decoder encounters end of the input source, engine will try to request next track via this method. If new correct url is provided, than engine will pre-buffer new data, so track switching will be smooth. If nil or incorrect url is provided, than engine will switch to the stop state at the end of the current track.
