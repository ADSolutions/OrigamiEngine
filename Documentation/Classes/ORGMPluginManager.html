Overview

The ORGMPluginManager provides transparent access to the supported decoders, sources and containers. All supported plugins are conform to the corresponding protocol, thus you have common protocol among same type plugins.

decoderForSource:error:

Factory method for decoder plugins.

- (id<ORGMDecoder>)decoderForSource:(id<ORGMSource>)source error:(NSError **)error

Parameters

source

A source object used to determine corresponding decoder.

error

A pointer to an NSError object. You do not need to create an NSError object.

Return Value

A new decoder object or nil if corresponding plugin is not found.

Discussion

Factory method for decoder plugins.

Decoder class is determined from the source url extension.
